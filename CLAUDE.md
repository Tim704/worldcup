# Project Context: World Cup 2026 Fantasy Hub
## Tech Stack
- Frontend: Web Development (React/TypeScript)
- Backend: Rust & Python
- Infrastructure: Local-first development, deployment via scripts.

## Architectural Mandates
- Prioritize performance, zero-cost abstractions in Rust, and execution efficiency.
- Provide clean, heavily documented code snippets.
- Use the metric system exclusively for all measurements, UI elements, and logging (e.g., margins in pixels/cm, weights in kg, temperatures in Celsius).
- For complex algorithmic problems, utilize a Chain of Thought derivation in comments before outputting the final function.
